# Integrated development environment

An integrated development environment (IDE) provides check-out and check-in capabilities to the [source code management (SCM)](./scm.md) system. Typically, it supports the language and coding practices of the developer, enables building the modified code within the sandbox, and drives unit tests as part of the coding phase.

The IBM Z DevOps solution is geared towards a "Bring Your Own IDE" approach, meaning we want to enable customers to use the IDE of their preference in our solution.

## Common IDE options for z/OS application development

- Eclipse-based: IBM Developer for z/OS (IDz)
- Visual Studio Code (VS Code) extension: IBM Z Open Editor (and IBM Z Open Debug)
- Browser-based: Wazi for Dev Spaces
